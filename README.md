# Santhosh P - Portfolio

A modern, responsive portfolio website built with React.js.

## Features

- Single-page application with smooth scroll navigation
- Responsive design for all devices
- Modern animations and transitions
- Dark theme with gradient accents

## Sections

- **Home** - Hero section with introduction
- **About** - Professional summary and highlights
- **Skills** - Technical skills and technologies
- **Experience** - Work history with project details
- **Education** - Academic background
- **Contact** - Contact information and social links

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm

### Installation

1. Clone the repository:

```bash
git clone https://github.com/SanthoshP85/MyPortfolio.git
cd MyPortfolio
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

To create a production build:

```bash
npm run build
```
